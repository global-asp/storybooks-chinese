Seve 头发

Clare Verbeek, Thembani Dladla, Zanele Buthelezi

Bronwen Heath, Ingrid Schechter

Shameem Oozeerally & MIE French Students

毛里裘斯克里奥尔语

1级别

本故事尚未有语音版。

Thuli so seve kourt.

图丽有一头短发。

Anna so seve long.

安娜有一头长发。

Cathy so seve pli long.

凯茜的头发最长。

Zama finn tres so seve.

萨玛把她的头发编成辫子。

Baba ena labarb.

巴巴留着胡子。

Zanele finn pengn so seve.

萨内蕾梳了头发。

Thabo finn koup so seve.

塔波剪了头发。

Themba finn raz so latet.

腾巴剃光了头发。

作者: Clare Verbeek, Thembani Dladla, Zanele Buthelezi
插图: Bronwen Heath, Ingrid Schechter
译文: Shameem Oozeerally & MIE French Students
语言: 毛里裘斯克里奥尔语
